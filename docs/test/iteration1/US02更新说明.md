# US02 功能更新说明

日期：2026-04-10

## 本次更新实现内容

本次完成了 US02「TA 上传简历」功能，主要包括：

- 在 TA 的个人资料页面新增简历上传区域，支持选择并上传文件。
- 支持的简历文件类型为：`PDF`、`DOC`、`DOCX`。
- 系统只保留 TA 最近一次上传的简历（新文件会覆盖旧文件）。
- 页面可显示当前已上传的简历文件名，方便 TA 确认上传结果。
- 当上传文件类型不符合要求时，系统会显示明确错误提示：仅允许上传 PDF 或 DOC/DOCX。
- 上传功能加入权限控制，仅 TA 角色可以执行简历上传操作。

## 更新效果

通过本次更新，TA 可以在申请岗位前维护并更新自己的最新简历，模块负责人在后续审核申请时可基于最新简历信息进行评估。

## 测试记录
- 当前简历上传功能在profile编写页面内，目的是方便后续通过建立填写profile
- 上传后的简历保存在uploads文件夹内，以cv_用户id命名，同一类型文件只允许保留一份
- 上传文件时主动过滤不匹配的文件类型，主动避免用户传入不正确类型的文件
